Python and Machine Learning Based Projects

Predicting House Prices: Machine Learning for Real Estate Analysis

I worked on this task of creating a house price prediction model using Random Forest Algorithm. The project was centered on the basics of Machine Learning and utilized a dataset from Kaggle containing information about houses in Delhi. The aim was to develop a model capable of accurate predictions.

  • Timeline: Jan. 2023 - April 2023

Forecasting Tomorrow's Temperature: The Regression Model Approach

I worked on a project to predict tomorrow's temperature using data up until today. The model is based on Machine Learning's Regression Model and performs best with the Ridge Regression Model. The project uses a dataset of the weather of Lucknow from 1990 till 2022.

  • Timeline: June 2023 - July 2023

Raspberry Pi 3 GUI for Connecting to Nearby Bluetooth Devices

Continuing from the OELP project (Course Project) on Software Stack for IoT devices, I took the initiative to develop a graphical user interface (GUI) using the Python tkinter library. This GUI was specifically designed to run on a Raspberry Pi 3, and it facilitated the connection to nearby Bluetooth devices by utilizing the Bluetooth capabilities of the Pi through Python programming.

  • Timeline: May 2023 - July 2023
